# 🌦️ Weather Now

A lightweight web application that allows users to search for any city and view real-time weather information.

---

## 🚀 Features

* Search weather by city name
* Displays:

  * Temperature
  * Feels like temperature
  * Humidity
  * Weather description
* Minimal and clean user interface

---

## 🛠️ Tech Stack

* JavaScript (ES6)
* HTML5
* CSS3
* Webpack

---

## 📂 Project Structure

```
src/
  components/     # UI logic
  models/         # Data models
  services/       # API calls
  style.css       # Styling
  index.js        # Entry point
  template.html   # HTML template
```

---

## ⚙️ Setup & Installation

1. Clone the repository

```
git clone https://github.com/Tejanshu9/weather-now.git
cd weather-now
```

2. Install dependencies

```
npm install
```

3. Run the development server

```
npx webpack serve
```

---

## 🌐 Deployment

To deploy using GitHub Pages:

```
npm run deploy
```
